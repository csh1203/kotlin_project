## 양의지 ⭐

## commit convention
|Tag Name|Description|
|---|---|
|Feat	| 기능 구현 |
|Fix	| 버그 수정 |
|Design	| 디자인 속성 변경 |
|!BREAKING CHANGE	| 엄청나게 큰 변화 |
|!HOTFIX | 급하게 수정 |
|Style | 코드 포맷 변경, 세미 콜론 누락, 코드 수정이 없는 경우|
|Refactor	| 코드 리팩토링 |
|Comment | 주석 추가 및 변경|
|Docs	| 문서 수정 |
|Test	| 테스트 코드 |
|Rename	| 파일 혹은 폴더명을 수정하거나 옮기는 작업 |
|Remove	| 파일을 삭제하는 작업 |
